This is a sample project to showcase how 3D models are presented on this site.

### How it was made

I used **Blender** to model the geometry and **Substance Painter** for the textures. The goal was to achieve a low-poly aesthetic while maintaining high-quality materials.

### Features

- Interactive 3D preview
- Responsive layout
- Support for AR viewing on mobile devices
