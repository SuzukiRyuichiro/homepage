### [Section L](https://section-l.co/)

##### 正社員: 2023年10月 - 2024年10月 / 業務委託: 2024年10月 - 2025年6月

##### プロジェクトマネージャー & フルスタックエンジニア {{< mdi alias="vuejs" >}} {{< mdi alias="language-javascript" >}} {{< mdi alias="language-ruby" >}} {{< mdi alias="language-ruby-on-rails" >}} {{<mdi alias="aws" >}}{{< mdi alias="lambda" >}}{{< devicon alias="dynamodb-plain" >}}

Section Lでの2度目の在籍期間中、フルスタックエンジニアとプロジェクトマネージャーを兼任し、要件定義からデリバリーまで複数の重要プロジェクトを主導しました。主な実績として、Section L 西新橋のサウナ施設予約システムをゼロから開発し、既存のInterSectionアプリケーションに統合しました。従来のRuby on Railsバックエンドとは別に、AWS（Lambda、DynamoDB）を活用したモダンなサーバーレスアーキテクチャを設計することで、高いスケーラビリティとパフォーマンスを実現しました。このシステムの導入により、予約の約70%がアプリから直接行われるようになり、スタッフの事務作業負担の大幅な削減に貢献しました。

また、基幹システム（PMS）の移行に伴う大規模なアーキテクチャ更新も従事しました。AWS LambdaとSQSを用いた既存バックエンドとのアダプターを設計・実装し、複数のアプリケーションと新システム間の複雑なAPI連携を管理しました。この過程でシニアエンジニアと密に連携してシステム設計を磨き上げ、関心の分離（Separation of Concerns）を徹底し、全サービス間でのデータ整合性を維持しました。

### [OpenSalary](https://opensalary.jp/) / [WorkCircle](https://workcircle.app)

##### 2022年6月 - 2023年8月

##### フルスタックソフトウェアエンジニア {{< mdi alias="docker" >}} {{< mdi alias="react" >}} {{< mdi alias="vuejs" >}} {{< mdi alias="language-javascript" >}} {{< mdi alias="language-ruby" >}} {{< mdi alias="language-ruby-on-rails" >}}

WorkCircle/OpenSalaryのスタートアップチームに参画し、匿名キャリアネットワークプラットフォームの製品開発とパフォーマンス最適化に注力しました。WorkCircleのネイティブアプリ（Ruby on Rails / React Native）の初期リリースに参画し、通知、投票機能、スレッド型のコメント機能などの主要なソーシャル機能を実装しました。技術的な成果として、メインフィードのクエリパフォーマンスを最適化し、ページ読み込み時間を約70%短縮し、ユーザーエクスペリエンスの直接的な向上に繋げました。

OpenSalaryでは、プロダクトマネージャーやデザイナーといった非エンジニア職種へのサービス拡大に向け、データ可視化とアップロードフローの再設計に参画しました。また、年収データのSNS共有ツールや新規ユーザーへの情報提供プロンプトなど、エンゲージメント向上のための機能を多数導入しました。特にナビゲーションバーの検索機能の刷新は、サイト内で最も利用される機能となり、情報の探しやすさを大幅に改善しました。
